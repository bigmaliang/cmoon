1: Linux文件权限及修改方法; 软链接与硬链接的创建方法、及区别
    文件权限：所有者、同组用户、其他用户分别可具有读、写、执行权限。可用chmod命令修改权限。
    软链接用ln -s 创建，硬链接用ln 创建。 删掉软链接的目标文件（或原文件）将会使软链接无法访问，删掉硬链接的则不会。


2: 写一个去掉字符串开始和结尾特定字符的函数(可修改输入字符串, 如:输入 "aa newa word aaa", 'a'; 输出 " newa word ")：
    char* strstripc(char *s, char c)
    {
        int x;

        x = strlen(s) - 1;
        while (x >= 0 && s[x] == c) s[x--] = '\0';

        while (*s && *s == c) s++;

        return s;

    }

5: 请描述socket通信常用的两种协议及其区别，以及同步、异步socket通信的区别及应用场景
    UDP, TCP 为常用的两种socket协议。
    同、异步socket是指程序中socket请求函数是否阻塞当前程序。
    同步socket主要用于需要确认网络请求是否成功，而异步socket主要用在对网络请求结果无需立即确认且并发较高的情况。
